## 🚀 Quick Start

```typescript
import { quickStart } from '@agentledger/suil';

// Initialize SUIL
const { engine, demo } = await quickStart();

// Run comprehensive demo
const results = await demo();
console.log(`Achieved ${results.performance.averageOpsPerSec} ops/sec!`);

// Process tasks with character-driven intelligence
const task = {
  id: 'example_task',
  type: 'cache_operation',
  input: { operation: 'set', key: 'demo', value: 'Hello SUIL!' },
  character: 'chihiro' // Performance-optimized personality
};

const result = await engine.processTask(task);
console.log(`Processed in ${result.processingTime}ms via ${result.route}`);
```
